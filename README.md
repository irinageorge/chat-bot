### Installation

1. Clone the repo:

```bash
git clone https://github.com/your-username/chat-bot-1-0.git
cd chat-bot-1-0
```

2. Install dependencies:

```bash
npm install
```

3. Copy and update your API keys in the `.env` file:

```bash
cp .env.example .env
```

4. Start the dev server:

```bash
npm run dev
```
